## 📖 About

**WishList** is a lightweight mobile application built as a personal learning project to explore **Kotlin**, **Android Studio**, and **Jetpack Compose**.

The core functionality revolves around managing a personalized gift list, with a clear separation between **admin** and **client user** roles:

- The **admin** can:
  - Add new items
  - Filter by category/type
  - Mark items as favorites
  - Search through the list using a dynamic bar
  - Remove existing items


- The **client user** can:
  - View items in real-time as they are added
  - Receive an **email notification** whenever a new product is published
  - See detailed product cards
  - Access external product URLs
  - Filter items by favorites or category
  - Search items using a dynamic search bar

### 🚀 Technologies Used

- **Jetpack Compose** – for modern UI development in Android.
- **Kotlin** – for robust and concise Android programming.
- **Firebase Authentication** – to handle user sign-in securely.
- **Cloud Firestore** – for real-time database and sync.
- **Brevo (formerly Sendinblue)** – to send transactional email notifications.
- **Git** – for version control.
- **GitHub** – as repository hosting and collaboration platform.
- **Android Studio** – Android SDK Tools, development.
